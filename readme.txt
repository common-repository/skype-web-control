Have a free Skype live chat and/or call window on your website to answer your visitor's question as fast as possible.

== Description ==
Answer your visitor's questions faster.

We've made it faster and easier for people to reach you straight from your site. Skype Web Control puts a Skype chat & call window into any website. It is easy to integrate and supports both desktop and mobile sites.

== Installation ==
1. Download and install the plugin from WordPress dashboard.
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Enter your Skype ID or if you have a bot, the Microsoft App ID.
4. Customize the colors to your liking
5. Click on "Save"
